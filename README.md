# Interns Tracker / Exit Portal

### Project Overview
The **Interns Tracker / Exit Portal** is a Streamlit-based web application that allows organizations to **track interns and applicants based on their current status**. Statuses can include Leave, Offline, Sleep, or other custom reasons. The dashboard provides a **centralized view** for easy monitoring and management.

### Tech Stack
- **Languages:** Python  
- **Libraries & Frameworks:** Streamlit, Pandas
